# 1st Blood - UI Design Tool

## Project Level: Starter (Next.js + Tailwind)

## Overview
A visual UI design tool where users can drag & drop elements to create designs.
Target: public-facing app for other users.

## Tech Stack
- Next.js 14+ (App Router)
- Tailwind CSS
- TypeScript

## Conventions
- Use `src/app/` for pages (App Router)
- Use `src/components/` for reusable components
- Use `src/components/ui/` for base UI primitives
- Component files: PascalCase (e.g., `Canvas.tsx`)
- Utility files: camelCase (e.g., `useCanvas.ts`)

## Key Features (Planned)
- Canvas with drag & drop
- Element creation (shapes, text, images)
- Property editing panel
- Export designs
