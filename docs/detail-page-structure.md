# 콘텐츠 상세페이지 구조도

## 전체 레이아웃

```
┌─────────────────────────────────────────────────────┐
│                    Header / GNB                     │
│  [로고]    [홈] [카테고리] [검색]    [로그인/프로필] │
├─────────────────────────────────────────────────────┤
│                  Breadcrumb 네비게이션               │
│  홈 > 카테고리 > 현재 글 제목                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌───────────────────────────┐  ┌────────────────┐  │
│  │                           │  │                │  │
│  │      메인 콘텐츠 영역      │  │   사이드바      │  │
│  │                           │  │                │  │
│  │                           │  │                │  │
│  └───────────────────────────┘  └────────────────┘  │
│                                                     │
├─────────────────────────────────────────────────────┤
│                    Footer                           │
└─────────────────────────────────────────────────────┘
```

---

## 1. Header (GNB)

```
┌─────────────────────────────────────────────────────┐
│ [로고]                                              │
│                                                     │
│ [홈]  [카테고리▼]  [인기글]  [태그]                  │
│                                                     │
│                        [🔍 검색]  [알림🔔] [프로필]  │
└─────────────────────────────────────────────────────┘
```

- 로고: 홈으로 이동
- 메인 네비게이션: 홈, 카테고리, 인기글, 태그
- 유틸리티: 검색, 알림, 프로필/로그인

## 2. 메인 콘텐츠 영역

### 2-1. 글 헤더

```
┌─────────────────────────────────────────────┐
│  [카테고리 배지]                             │
│                                             │
│  글 제목 (H1)                               │
│  ─────────────────────────────              │
│                                             │
│  [프로필 이미지]  작성자명                    │
│                  2026.03.10 · 조회 1,234     │
│                  읽기 약 5분                  │
│                                             │
│  [#태그1]  [#태그2]  [#태그3]                │
│                                             │
│  ──────────────────────────────────────     │
└─────────────────────────────────────────────┘
```

**구성 요소:**
- 카테고리 배지 (라벨)
- 글 제목 (H1, 최대 2줄)
- 작성자 정보: 프로필 이미지 + 이름 + 작성일 + 조회수
- 예상 읽기 시간
- 태그 목록

```
│                                             │
│                                             │
```

---

### 2-3. 본문 콘텐츠

```
┌─────────────────────────────────────────────┐
│                                             │
│  본문 텍스트 (리치 텍스트)                    │
│                                             │
│  - H2, H3 소제목                             │
│  - 본문 텍스트 (p)                           │
│  - 굵기, 기울임, 밑줄, 취소선                 │
│  - 순서형/비순서형 리스트                     │
│  - 인용문 블록 (blockquote)                  │
│  - 코드 블록 (syntax highlighting)           │
│  - 인라인 이미지 + 캡션                      │
│  - 임베드 (YouTube, 트윗 등)                 │
│  - 구분선 (hr)                              │
│  - 링크 (내부/외부)                          │
│  - 테이블                                   │
│  - 콜아웃 / 알림 박스 (info, warning, tip)   │
│                                             │
└─────────────────────────────────────────────┘
```

**지원 콘텐츠 타입:**
| 타입 | 설명 |
|------|------|
| 텍스트 | 기본 본문, 제목(H2~H4), 인라인 스타일 |
| 미디어 | 이미지, 동영상 임베드, GIF |
| 코드 | 인라인 코드, 코드 블록 (언어별 하이라이팅) |
| 구조 | 리스트, 테이블, 인용문, 구분선 |
| 특수 | 콜아웃 박스, 접기/펼치기(아코디언), 각주 |

### 2-4. 글 하단 액션 바

```
┌─────────────────────────────────────────────┐
│                                             │
│  [❤️ 좋아요 42]  [🔖 북마크]  [🔗 공유▼]     │
│                                             │
│  ──────────────────────────────────────     │
│                                             │
│  [#태그1]  [#태그2]  [#태그3]                │
│                                             │
│  [◀ 이전글: 이전 글 제목...]                 │
│  [다음글: 다음 글 제목... ▶]                 │
│                                             │
└─────────────────────────────────────────────┘
```

**구성 요소:**
- 좋아요 (토글 + 카운트)
- 북마크/스크랩 (토글)
- 공유: 링크 복사, SNS 공유 (카카오, 트위터, 페이스북)
- 태그 (재표시, 클릭 시 태그별 목록으로 이동)
- 이전글 / 다음글 네비게이션

```

│  ──────────────────────────────────────     │
│                                             │
│  ┌─ 댓글 입력 ─────────────────────────┐    │
│  │ [프로필]  댓글을 작성해주세요...      │    │
│  │                          [등록]     │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  정렬: [최신순 ▼]                            │
│                                             │
│  ┌─ 댓글 아이템 ───────────────────────┐    │
│  │ [프로필]  닉네임  ·  2시간 전        │    │

│  │ [❤️ 3]  [답글]  [신고]  [···]        │    │
│  │                                     │    │
│  │  ┌─ 답글 ────────────────────┐      │    │
│  │  │ [프로필]  닉네임  · 1시간 전│      │    │

│  [댓글 더보기 ▼]                             │
│                                             │
```

**댓글 기능:**
- 댓글 작성 (로그인 필요)
- 대댓글 (1depth)
- 좋아요, 신고
- 정렬: 최신순 / 인기순
- 더보기 (페이지네이션 or 무한 스크롤)
- 수정 / 삭제 (본인 댓글)

## 3. 사이드바 (데스크탑)

```
┌────────────────────┐
│  📋 목차 (TOC)      │
│  ─────────────     │
│  1. 소제목1         │
│  2. 소제목2         │
│     2-1. 하위      │
│  3. 소제목3         │
│                    │
│  (스크롤 따라 고정)  │
├────────────────────┤
│                    │
│  📌 관련 글         │
│  ─────────────     │
│  [썸네일] 제목1     │
│  [썸네일] 제목2     │
│  [썸네일] 제목3     │
│                    │
├────────────────────┤
│                    │
│  🏷️ 인기 태그       │
│  ─────────────     │
│  [태그1] [태그2]    │
│  [태그3] [태그4]    │
│                    │
├────────────────────┤
│                    │
│  📢 광고/배너       │
│  (선택적 영역)      │
│                    │
└────────────────────┘
```

**구성 요소:**
- **목차 (TOC)**: H2/H3 기반 자동 생성, 스크롤 위치 하이라이팅, sticky
- **관련 글**: 같은 카테고리/태그 기반 추천 3~5개
- **인기 태그**: 사이트 전체 인기 태그
- **광고/배너**: 선택적

### 모바일 (< 768px)

```
┌───────────────────────┐
│  [≡]  [로고]  [🔍][👤] │
├───────────────────────┤
│ 홈 > 카테고리          │
├───────────────────────┤
│ [카테고리]             │
│                       │
│ 글 제목 (H1)          │
│                       │
│ [프로필] 작성자        │
│ 2026.03.10 · 조회수   │
├───────────────────────┤
│                       │
│ [커버 이미지 (Full)]   │
│                       │
├───────────────────────┤
│                       │
│ 본문 콘텐츠            │
│ (전체 너비)            │
│                       │
├───────────────────────┤
│ [❤️] [🔖] [🔗]  [목차📋]│
│ (하단 플로팅 바)       │
├───────────────────────┤
│ 작성자 프로필 카드     │
├───────────────────────┤
│ 추천 콘텐츠 (세로 슬라이드)│
├───────────────────────┤
│ 댓글 섹션              │
├───────────────────────┤
│ Footer                │
└───────────────────────┘
```

**모바일 변경사항:**
- 사이드바 → 숨김 (목차는 플로팅 버튼으로 접근)
- 액션 바 → 하단 고정 플로팅
- 추천 콘텐츠 → 가로 스크롤 카드
- 햄버거 메뉴로 GNB 축소

## 8. 주요 API 엔드포인트 (예상)

| 엔드포인트 | 메서드 | 설명 |
|-----------|--------|------|
| `/api/posts/:id` | GET | 게시글 상세 조회 |
| `/api/posts/:id/like` | POST/DELETE | 좋아요 토글 |
| `/api/posts/:id/bookmark` | POST/DELETE | 북마크 토글 |
| `/api/posts/:id/comments` | GET | 댓글 목록 조회 |
| `/api/posts/:id/comments` | POST | 댓글 작성 |
| `/api/comments/:id` | PUT/DELETE | 댓글 수정/삭제 |
| `/api/comments/:id/like` | POST/DELETE | 댓글 좋아요 |
| `/api/posts/:id/related` | GET | 관련 글 조회 |
| `/api/posts/:id/views` | POST | 조회수 증가 |
